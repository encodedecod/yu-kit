## Features

- 🚀 高质量组件，覆盖主流场景
- 🚀 集工具函数，hooks，组件库于一身，分包开发
- 🍭 支持按需引用
- 💪 支持 TypeScript

## Install

项目推荐使用 pnpm 的方式安装：

1. 安装依赖

组件库#

```bash
pnpm add @yu-kit/components
```

hooks 库#

```bash
pnpm add @yu-kit/hooks
```

工具库#

```bash
pnpm add @yu-kit/kit
```

函数库#

```bash
pnpm add @yu-kit/utils
```
