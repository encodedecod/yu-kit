# 快速开始

## 安装

### 组件库

```bash
 npm i @yu-kit/components
```

### hooks 库

```bash
 npm i @yu-kit/hooks
```

### 工具库

```bash
 npm i @yu-kit/kit
```

### 函数库

```bash
 npm i @yu-kit/utils
```

## 简单使用

```ts
import { ElementHandler } from '@yu-kit/kit'
import { useLoading } = from '@yu-kit/hooks'
import { YuButton } = from '@yu-kit/components'
import { interval } = from '@yu-kit/utils'
```
